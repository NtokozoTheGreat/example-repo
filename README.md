# Example Repo

## Project Name
Album Management Script

## Description
This project contains a Python script  that defines a class  to represent music albums. 
It allows you to:

- Create Album objects with name, number of songs, and artist.
- Store albums in lists and manipulate them.
- Print a readable list of albums.
- Sort albums by number of songs or alphabetically by name.
- Search for an album by name interactively.

The script demonstrates object-oriented programming, list manipulation, sorting, and searching in Python.
